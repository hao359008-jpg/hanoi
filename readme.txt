=======================================================
              圣杯战争（汉诺塔版） - 项目说明文档
=======================================================

一、项目概述
-------------------------------------------------------
本项目是一款基于Fate系列主题的汉诺塔游戏，包含两个版本：
1. C++控制台版本 - 原生桌面应用
2. Web网页版本 - 支持在线访问

游戏特色：
- Fate系列主题视觉设计
- 支持3-8层难度选择
- 自动演示功能
- 个人最佳记录保存
- 计时器功能
- 撤销功能

=======================================================

二、运行环境要求
-------------------------------------------------------

【C++控制台版本】
-----------------
- 操作系统：Windows 7/10/11 (64位)
- 编译器：GCC 7.0+ 或 MSVC 2017+

【Web网页版本】
---------------

- 服务器：任意HTTP服务器（推荐Netlify/Vercel/GitHub Pages）
- 无需额外依赖，纯前端实现

四、C++版本编译与运行
-------------------------------------------------------

【编译命令】
cd d:\c++实践
g++ -std=c++17 *.cpp -o hanoi.exe -Wall

【运行方式】
直接双击 hanoi.exe 或命令行运行：
hanoi.exe

【注意事项】
- 确保所有.cpp和.h文件在同一目录
- 若出现权限错误，先关闭运行中的进程：
  taskkill /f /im hanoi.exe

六、游戏操作说明
-------------------------------------------------------

【游戏规则】
1. 将所有盘子从1号柱移动到3号柱
2. 每次只能移动一个盘子
3. 大盘子不能放在小盘子上面
4. 最优步数：2^n - 1（n为层数）

【控制台版本操作】
-------------------
主菜单：
  1. 开始游戏
  2. 自动演示
  3. 设置
  4. 保存游戏
  5. 加载游戏
  0. 退出

游戏中：
  输入格式：from to (如 "1 2" 表示从1号柱移到2号柱)
  输入 "q" 退出游戏
  输入 "r" 重置游戏

【网页版本操作】
-----------------
1. 点击选择难度（3-8层）
2. 点击柱子选中/移动盘子
3. 功能按钮：
   - 重新开始：重置游戏
   - 自动演示：自动播放最优解法
   - 撤销一步：回退上一步
   - 游戏说明：显示规则
4. 右上角音乐按钮：控制背景音乐

八、输入输出格式
-------------------------------------------------------

【控制台输入】
移动命令：[来源柱] [目标柱]
示例：1 2 (将1号柱顶部盘子移到2号柱)

【控制台输出】
游戏状态显示格式：
        =1=        
       ==2==       
      ===3===      
    ────────────  1号柱
    ────────────  2号柱
    ────────────  3号柱
步数: 0 | 最优: 7

【网页输出】
- 实时显示步数、时间、最佳记录、最优步数
- 胜利时弹出"契约达成！"提示框

九、常见问题
-------------------------------------------------------

Q: Web版本图片不显示？
A: 确保所有图片与index.html在同一目录，使用相对路径

Q: 音乐无法播放？
A: 确保bgmusic.mp3存在，点击页面后再开启音乐

Q: C++版本编译失败？
A: 检查是否所有源文件都在目录中，使用g++ *.cpp编译

Q: 自动演示无法停止？
A: Web版本点击"自动演示"后会出现停止按钮
